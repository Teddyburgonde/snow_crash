on a vue un decalage donc on a creer un programme qui stop le decalage

```c
#include <stdio.h>

int main(int argc, char **argv)
{
    int i;

    i = 0;

    if (argc != 2)
        return 1;
    while (argv[1][i])
    {
        printf("%c", argv[1][i] -i);
        i++;
    }
}
```
